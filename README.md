## 📌 Overview

This project presents an interactive Power BI dashboard built using Blinkit grocery sales data. The dashboard provides insights into sales performance, customer trends, and product analysis through visualizations.

It helps in understanding business performance and supports data-driven decision making.

---

## 🎯 Objective

To design and develop an interactive dashboard that visualizes sales data and provides actionable insights.

---

## 🧩 Key Areas Covered

• Sales performance analysis
• Product category insights
• Customer behavior analysis
• Revenue trends visualization
• Interactive filtering and slicing

---

## 📊 Dataset

The dataset contains Blinkit grocery sales data including product details, sales, and customer-related information.

• BlinkIT Grocery Data.xlsx

## 📈 Dashboard Features

• Monthly/Yearly trends
